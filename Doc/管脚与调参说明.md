# 管脚与调参说明

## 一、硬件管脚

| 模块 | STM32 管脚 | 说明 |
|---|---|---|
| HC-SR04 TRIG | PB14 | 前方超声波触发 |
| HC-SR04 ECHO | PB15 | 前方超声波回波；PB15 为 FT 数字输入，可选分压保护 |
| 左红外 | PA12 | 检测到障碍物为低电平 |
| 右红外 | PA11 | 检测到障碍物为低电平 |
| USART1 TX | PA9 | 接 USB-TTL 的 RX |
| USART1 RX | PA10 | 接 USB-TTL 的 TX，当前预留接收 |
| L298N IN1 | PB6 | 左电机方向控制 |
| L298N IN2 | PB7 | 左电机方向控制 |
| L298N IN3 | PB8 | 右电机方向控制 |
| L298N IN4 | PB9 | 右电机方向控制 |
| L298N ENB | PA1 / TIM2_CH2 | 右电机 PWM |
| L298N ENA | PA2 / TIM2_CH3 | 左电机 PWM |

HC-SR04 必须按以下方式连接：

```text
HC-SR04 VCC  -> 5V
HC-SR04 GND  -> STM32 GND（同时与 L298N、电机电源共地）
HC-SR04 TRIG -> PB14
HC-SR04 ECHO -> PB15
```

建议在 ECHO 与 PB15 之间加分压保护：ECHO 到 PB15 使用 1 kΩ，PB15 到 GND 使用 2 kΩ。调试时应在超声波正前方 10~30 cm 放置平整硬质物体。

## 二、红外电平

本工程已经设置为：检测到障碍物 = 低电平。

```c
#define IR_BLOCKED_LEVEL 0
```

代码位置：`USER/config.h`。

## 三、PWM 调节

所有 PWM 值均在 `USER/config.h` 中集中修改。本版已把差速驱动 PWM 调低：

```c
#define MOTOR_FORWARD_PWM         800
#define MOTOR_SLOW_PWM            720
#define MOTOR_TURN_INNER_PWM      600
#define MOTOR_TURN_OUTER_PWM      800
#define MOTOR_TURN_BACK_INNER_PWM 600
#define MOTOR_TURN_BACK_OUTER_PWM 800
```

本版转弯使用差速转弯，左转时右轮快、左轮慢；右转时左轮快、右轮慢。两个轮子都向前转，不采用正反转原地转弯。

## 四、闭环转向调节

```c
#define MOTOR_FEEDBACK_TURN_PWM           360
#define TURN_CREEP_PULSE_MS               35U
#define TURN_CREEP_SETTLE_MS              30U
#define TURN_FEEDBACK_DISTANCE_CHANGE_CM  8U
#define TURN_FEEDBACK_STABLE_SAMPLES      2U
#define TURN_FEEDBACK_MAX_SAMPLES         60U
#define TURN_FEEDBACK_MAX_INVALID_SAMPLES 2U
```

转向不再按固定总时间结束。车辆采用“短脉冲旋转 -> 停车等待稳定 -> 读取传感器”的蠕动方式；检测到新方向前方可通后，车辆保持停车并再次采样确认，不会为了确认状态额外再转一步。`TURN_CREEP_PULSE_MS` 越小，每一步转动越细，通常更稳定但转弯更慢；建议从 `20~40 ms` 调节。最大采样次数仅用于传感器异常或完全开放区域时的安全停车，不用于决定正常转向角度。

只有两路侧向数字红外和一个前向超声波时，在十字路口或无墙开放区域没有足够参照，无法保证精确 90°。此时程序输出 `ACT=TURN_FEEDBACK_FAIL` 并停车。若需要在任何场景下精确转角，需要增加轮编码器或陀螺仪。

## 五、算法说明

迷宫算法位于 `APP/maze.c`。

核心逻辑已改为左手原则：

```text
左侧无障碍 -> 左转
前方安全 -> 前进
右侧无障碍 -> 右转
否则 -> 差速左转掉头
```

## 六、USART1 串口状态输出

- PA9：USART1_TX，接 USB-TTL 的 RX。
- PA10：USART1_RX，接 USB-TTL 的 TX，当前预留接收。
- 串口参数：115200-8-N-1。
- 输出格式：`US=18cm,US_OK=1,US_ERR=NONE,IR_L=BLOCK,IR_R=CLEAR,PWM_L=+600,PWM_R=+800,ACT=LEFT_TURN`。
- `US_OK=0` 表示超声波本次测距失败，车辆会停止并等待下一次检测。
- `US_ERR=ECHO_IDLE_HIGH`：触发前 ECHO 一直为高电平，检查 ECHO 是否接错、短接到 5V 或模块损坏。
- `US_ERR=NO_ECHO_RISE`：触发后 ECHO 一直没有高电平，重点检查模块 5V 供电、共地、PB14/TRIG、PB15/ECHO 是否接反。
- `US_ERR=ECHO_HIGH_TIMEOUT`：ECHO 拉高后一直不恢复，检查 ECHO 接线或模块故障。
- `US_ERR=TOO_CLOSE`：收到回波，但目标距离小于 2 cm。
- `US_ERR=TOO_FAR`：收到回波，但目标距离大于 300 cm；闭环转向时会将其视为前方开阔。
- `IR_L` 和 `IR_R` 分别表示左右红外状态：`BLOCK` 为检测到障碍物，`CLEAR` 为无障碍。
- `PWM_L` 和 `PWM_R` 分别表示左右电机实际 PWM。正数为前进，负数为后退，`0` 为停止。
- `ACT` 表示当前控制动作；`US_FAULT_STOP` 表示因为超声波无效而停车；`TURN_ALIGNED` 表示闭环转向完成；`TURN_FEEDBACK_FAIL` 表示转向过程中没有找到可靠方向参照。
- 输出周期：约 0.3 s。
- 开关、波特率和输出周期在 `USER/config.h` 中调整：

```c
#define UART_STATUS_ENABLE        1
#define UART1_BAUDRATE            115200U
#define UART_STATUS_INTERVAL_MS   300U
```

注意：USB-TTL 与 STM32 必须 GND 共地。

正式迷宫模式中，超声波连续三次无效后会输出 `ACT=US_FAULT_STOP` 并停止电机，避免传感器断线时车辆失控。若要排除电机硬件问题，可直接烧录 `Objects/MazeRobot_motor_test.hex`；该固件不依赖超声波有效状态，会循环测试前进、后退、左转和右转。

## 七、HC-SR04 电平保护

HC-SR04 通常使用 5V 供电，其 ECHO 输出约为 5V。STM32F103 的 PB15 属于 FT（5V tolerant）数字输入，因此可以直接连接。若使用来源不明的兼容板或希望增加保护，也可以增加分压电阻或电平转换模块。
